SqueezeCloud plugin for Logitech Media Server

Point your server at http://grobbie.github.io/SqueezeCloud/public.xml

Good to know:

You'll need OpenSSL development headers on your server for this plugin or it won't work
On debian that's

sudo apt-get install libssl-dev
sudo perl -MCPAN -e 'install IO::Socket::SSL'
sudo service logitechmediaserver restart

That is all.
